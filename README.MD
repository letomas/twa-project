# Semestral work TWA.1 - booking system

## Dependencies
- Php 7.4
- Composer
- Doctrine
- Symfony

## Setup
1. Download/Clone repository
2. Install dependencies with composer. Use IDE or command line:
```
php composer.phar install
```
3. Run in the console `symfony server:start`
